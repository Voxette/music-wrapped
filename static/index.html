How do I get an API key?
1. Go to last.fm/api/account/create
2. Sign in (or create a free Last.fm account)
3. Fill in any app name (e.g. "Music Wrapped") and description
4. Leave the callback URL blank
5. Click "Submit" — your API key will be shown on the next page
6. Copy the API Key (not the shared secret) and paste it above

Used to fetch scrobbles (Last.fm username mode) and genre tags for all sources. Your key stays in your browser and is sent only to Last.fm's API.

How do I get my Spotify data export?
1. Go to spotify.com/account/privacy
2. Scroll to "Download your data"
3. Check "Extended streaming history" (uncheck "Account data" — you don't need it)
4. Click "Request data"
5. Check your email — Spotify sends a confirmation link you must click to start the export
6. Wait for Spotify to email you again when it's ready (usually 5-30 days)
7. Download the ZIP, then upload it here (or extract and upload the JSON files)

The export contains files like Streaming_History_Audio_2020-2024_0.json
